What It Does

Prevent fan blast

Ramps earlier toward your quiet ceiling so heat is handled before the system needs a loud emergency response.

Steady quiet floor

Keeps a constant low fan floor while still allowing macOS to ask for more cooling whenever it needs to.

Manual and test controls

Lets you test audible RPM levels while the macOS baseline marker shows what the system is asking for.
